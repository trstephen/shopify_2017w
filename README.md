### Running the solution
Install the node version in [.nvmrc](.nvmrc).
[nvm](https://github.com/creationix/nvm) is cool and good.
```shell
git clone git@github.com:trstephen/shopify_2017w.git

cd shopify_2017w

# if nvm installed
nvm use

npm install

node index.js
```
